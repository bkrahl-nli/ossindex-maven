ossindex:audit

Full name:

org.sonatype.ossindex.maven:ossindex-maven-plugin:1-SNAPSHOT:audit

Description:

Vulnerability audit of project dependencies via Sonatype OSS Index.

Attributes:

  • Requires a Maven project to be executed.
  • Requires dependency resolution of artifacts in scope: test.
  • Since version: ???.

Optional Parameters

Name Type Since Description
<baseUrl> URI ??? Override Sonatype OSS Index service base-URL.
User property is: ossindex.baseUrl.
<clientConfiguration> OssindexClientConfiguration ??? Sonatype OSS Index client configuration.
<cvssScoreThreshold> float ??? CVSS-score threshold. Vulnerabilities with lower scores will be excluded.
Default value is: 0.
User property is: ossindex.cvssScoreThreshold.
<excludeCoordinates> Set ??? Set of coordinates to exclude from vulnerability matching.
<excludeCoordinatesCsv> String ??? Set excludeCoordinates from a comma-separated list.
User property is: ossindex.excludeCoordinates.
<excludeVulnerabilityIds> Set ??? Set of Sonatype OSS Index vulnerability identifiers to exclude from matching.
<excludeVulnerabilityIdsCsv> String ??? Set excludeVulnerabilityIds from a comma-separated list.
User property is: ossindex.excludeVulnerabilityIds.
<fail> boolean ??? When vulnerable components are found; fail the build.
Default value is: true.
User property is: ossindex.fail.
<reportFile> File ??? Export component-report to file. Supports .json, .xml and .txt extensions.
User property is: ossindex.reportFile.
<scope> String ??? Limit scope of dependency resolution.
User property is: ossindex.scope.
<skip> boolean ??? Skip execution.
Default value is: false.
User property is: ossindex.skip.
<transitive> boolean ??? Include transitive dependencies.
Default value is: true.
User property is: ossindex.transitive.
